School Management System in C
Overview

This project is a School Management System built in C language.
It allows users to add students, perform various operations, and query information about the system, such as:

Which student has the highest grades

Other statistics about students

Features

Add, remove, and manage student records

Query top-performing students

Efficient memory management using a fixed-size heap to minimize runtime complexity

Technologies Used

C Language for core implementation

Fixed-size heap for optimized performance
